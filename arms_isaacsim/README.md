# ARMS Isaac Sim Integration

Step-by-step pick-and-place simulation of the ARMS assembly sequence.

## Prerequisites

- Isaac Sim 6.0 virtual environment at `~/isaacsim_env`
- ARMS planner run to produce output in `assembler_output/`

## Install

```bash
python3 -m venv ~/isaacsim_env --prompt isaacsim
~/isaacsim_env/bin/pip install isaacsim[all,ros2]==6.0.0.0 \
    --extra-index-url https://pypi.nvidia.com
```

## Import the vacuum tool model (once you have the STEP/STL file)

The tool model must be oriented with the vacuum contact face at the model
origin, nozzle body extending in +Z.  Units: millimetres.

```bash
~/isaacsim_env/bin/isaacsim --headless \
    --exec arms_isaacsim/scripts/import_tool.py \
    --tool-file path/to/vacuum_tool.step
```

This writes `arms_isaacsim/assets/usd/vacuum_tool.usd`.  Until this file
exists, the scene uses a cylinder placeholder.

## Run the simulation (GUI)

```bash
cd /path/to/arms_ros2_ws
~/isaacsim_env/bin/isaacsim \
    --ext-folder arms_isaacsim/exts \
    --exec arms_isaacsim/scripts/run_simulation.py \
    --output-dir assembler_output
```

Opens Isaac Sim with the full scene loaded and a timeline animation covering
every pick-and-place step.  Scrub the timeline to step through the sequence.

## Run headless (e.g. render video)

```bash
~/isaacsim_env/bin/isaacsim \
    --ext-folder arms_isaacsim/exts \
    --exec arms_isaacsim/scripts/run_simulation.py \
    --output-dir assembler_output \
    --scene-usd /tmp/arms_scene.usd \
    --headless
```

## Directory structure

```
arms_isaacsim/
  exts/arms.assembly.sim/
    config/extension.toml          Isaac Sim extension manifest
    arms/assembly/sim/
      scene_builder.py             Converts STL/YAML output → USD stage
      sequence_player.py           Writes time-sampled gripper animation
  scripts/
    run_simulation.py              Main entry point
    import_tool.py                 One-off tool STEP/STL → USD converter
  assets/usd/
    vacuum_tool.usd                (generated by import_tool.py)
```

## Coordinate conventions

| System      | Units | Z = 0 meaning            |
|-------------|-------|---------------------------|
| ARMS / OCCT | mm    | ARMS workspace reference  |
| Jig STL     | mm    | jig base plate bottom     |
| USD scene   | m     | same as jig STL × 0.001  |

All positions in `assembly_plan.yaml` and `*_cache.yaml` are in mm.
`SequencePlayer` converts to metres before writing USD xformOps.
